## 4. 静态资源接入方式（audios / videos / voices）

### 4.1 资源目录建议（Flask 标准）
后端目录：
- `./static/videos/`
- `./static/audios/`
- `./static/voices/`（如需要存放不同说话人/音色资源，可新增）

访问路径（前端引用）：
- 视频：`/static/videos/xxx.mp4`
- 音频：`/static/audios/xxx.wav` 或 `/static/voices/xxx.wav`

### 4.2 前端访问静态资源的写法
在 Vue 页面中直接写：
- `<video :src="videoSrc">`
- `<audio :src="ttsSrc">`

其中 `videoSrc/ttsSrc` 最终应是 **浏览器可访问 URL**（推荐 `/static/...`）。

## 5. “重建运行代码”（开发常用）
- 修改前端代码后：
   ```bash
    cd frontend_vue 
    npm run build
   ```
- 将build在总项目目录里的4个HTML文件手动移动到templates文件夹中，否则运行app.py后将看不到前端变化。
